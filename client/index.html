ABOUT

This is an efficient implementation of a Convolutional Neural Network (CNN) including both back and forward propagation, written using only NumPy. I built this project to gain a deep understanding of the math behind CNNs, and to challenge myself more than I have before in Deep Learning. This demo uses a model trained on the EMNIST dataset to recognize 40 alphanumeric characters.
